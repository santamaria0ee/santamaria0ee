## **Proyectos Relevantes**

### **🔹 Sistema Integral de Gestión para Restaurante**

Sistema administrativo completo para control de operaciones:

* Gestión de pedidos, inventario y recetas
* Control de empleados y roles
* Conexión a base de datos MySQL
* Arquitectura MVC con separación de capas
* Impacto: Digitalización de procesos operativos y mejora en tiempos de servicio.

---

### **🔹 Sistema de Punto de Venta – Farmacia**

Aplicación desktop enfocada en ventas y control de productos:

* Registro de ventas en tiempo real
* Historial de operaciones
* Autenticación de usuarios
* Tecnologías: Java Swing, DAO, persistencia estructurada.

---

### **🔹 Agenda de Contactos con Persistencia Inteligente**

Aplicación modular con enfoque en buenas prácticas:

* CRUD completo
* Búsqueda optimizada
* Persistencia en archivos y base de datos
* Enfoque: arquitectura limpia y reutilización de componentes.

---

### **🔹 API RESTful de Gestión de Usuarios**

Backend con endpoints estructurados:

* CRUD de usuarios
* Serialización JSON
* Arquitectura en capas
* Tecnologías: Spring Boot, JPA.
